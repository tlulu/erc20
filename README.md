# ERC20 Implementation

A sample implementation of ERC20 using Foundry.

## Getting Started
Install Foundry
```
curl -L https://foundry.paradigm.xyz | bash
```

Install Foundry tools
```
foundryup
```

Run tests
```
forge test
```
